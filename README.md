THY-Twin

This repository provides the example code and data for the paper:

THY-Twin: A Digital Navigation and Auditing Twin for Ultrasound-Guided Thyroid Ablation

Overview

THY-Twin is a digital twin framework designed for ultrasound-guided thyroid ablation, supporting digital navigation and postoperative auditing.

This repository contains example code and related data associated with the paper.

Detailed instructions for installation, data preparation, and usage will be provided in a future update.
